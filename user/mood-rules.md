# 情绪-音乐映射规则

## 天气
- 晴天 → 轻快、明亮、流行
- 阴天 → 温暖、治愈、民谣
- 雨天 → 安静、深沉、慢歌
- 雪天 → 温暖、浪漫、爵士

## 时段
- 早晨 (06-09) → 元气唤醒，节奏明快
- 上午 (09-12) → 专注背景，Lo-fi / 纯音乐
- 中午 (12-14) → 轻松休息，轻流行
- 下午 (14-18) → 适度提神，R&B / Funk
- 傍晚 (18-21) → 放松转换，民谣 / 独立
- 深夜 (21-06) → 安静陪伴，慢歌 / 爵士

## 特殊场景
- 学习 → 纯音乐 / Lo-fi / 古典
- 运动 → 节奏感强 / 电子 / Hip-hop（但不能太吵）
- 心情不好 → 温暖治愈，不要 sad song 叠加
- 开心 → 分享喜悦的，可以推荐新风格探索
